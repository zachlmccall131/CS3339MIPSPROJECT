# CS3339MIPSPROJECT
CS 3339 MIPS Project spring 2026


The purpose of this project is to implement different MIPS instructions and demonstrate them.


I just pushed a new commit. Here is how to compile it.
First put in this command

g++ -std=c++11 main.cpp mips.cpp -o mips_simulator

Then simply compile with the assembly file

./mips_simulator input_file.asm

Enjoy.
